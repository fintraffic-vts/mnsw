## Finland-specific special notes

- MIG-P28 Cancellation mechanism: If a ship will not visit the port, related formalities can be cancelled by sending a COA.
- A COA means the ship call is cancelled and can only be reported by the declarant of the VID formality.
- Cancellation requests must be acknowledged by a formality response.
- After successful cancellation, no further formalities can be sent for the same Visit ID.
- Cancellation is final and cannot be withdrawn.
